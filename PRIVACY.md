# Adatvédelmi Irányelvek (Privacy Policy)

**Utolsó frissítés:** 2026. február 26.

A Trust Monitor ("Bővítmény") tiszteletben tartja és védi az Ön magánszféráját. Ez az irányelv elmagyarázza, hogyan kezeljük az adatait, követve a Privacy-First elveket és a Chrome Web Store Developer Program Policies iránymutatásait.

## Milyen adatokat gyűjtünk?
A bővítmény alapüzemmódjában **kizárólag a telepített Chrome bővítményekkel kapcsolatos metaadatokat** gyűjti (név, ID, verzió, engedélyek, domain hozzáférések). 
Ezeket az adatokat arra használjuk, hogy egy helyi kockázati elemzést ("Risk Score") futtassunk le.

## Hol tároljuk az adatokat?
A Bővítmény **MINDEN adatot kizárólag a saját gépén (Chrome Local Storage)** tárol. 
Semmilyen adat nem hagyja el a számítógépét, és egyetlen külső szerverre vagy felhőszolgáltatásba sem kerül továbbításra (No Telemetry, No Remote Servers).

## Mit NEM gyűjtünk?
Hangsúlyozzuk, hogy a Bővítmény:
* **Nem követi** nyomon az Ön böngészési előzményeit.
* **Nem fér hozzá** a meglátogatott weboldalak tartalmához (kivéve az explicit "Watch Mode" bekapcsolásánál, ahol is kizárólag a DOM struktúrára fut le egy szabályrendszer és abból soha semmilyen adat nincs elmentve vagy elküldve).
* **Nem ad el semmilyen adatot**, mivel fizikai hozzáférése sincs semmilyen külső szerverhez.

## Biztonság
A lokálisan tárolt adatokhoz csak a telepített böngésző és a Bővítmény férhet hozzá. A JSON alapú export ("Audit Log") funkció használatakor a felhasználó a saját íróasztalára mentheti ki a biztonsági jelentést, melyet saját belátása szerint oszthat meg.

## Kapcsolat
Amennyiben bármilyen kérdése lenne az adatvédelmi irányelvekkel kapcsolatban, kérjük, hozza létre a megfelelő Issue-t a projekt GitHub vagy támogatási oldalán.
